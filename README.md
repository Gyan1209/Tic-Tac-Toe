# Tic Tac Toe Game (JavaScript)

A browser-based Tic Tac Toe game built using **HTML, CSS, and Vanilla JavaScript**.  
The game supports two players, detects win and draw conditions, highlights the active player, and allows restarting the game without page reload.

---

## Features

- Two-player Tic Tac Toe gameplay (X vs O)
- Win detection using predefined winning conditions
- Draw detection when the board is full
- Visual highlight for the active player
- Restart game functionality without refreshing the page
- Clean and interactive UI

---

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **DOM Manipulation**
- **Event Handling**
